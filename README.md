## 💻 Sobre o projeto

Mochila de Viagem é um projeto para se possa anotar os itens para um viagem, o projeto conta com inclusão, modificação e exclusão de itens.

---

## ⚙️ Funcionalidades

- [x] Inclusão de itens;
- [x] Exclusão de itens;
- [x] Edição de itens;
- [x] Persistencia dos dados via Local Storage;
